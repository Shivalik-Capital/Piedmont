# Piedmont Roadmap

## Version 1: Core Market Dashboard

Establish the application foundation with live index data, backend APIs, and the Piedmont terminal design language.

## Version 2: Macroeconomic Dashboard

Add RBI rates, inflation, GDP, bond yields, USD/INR, gold, and crude oil.

## Version 3: Company Analytics

Add company search, financial statements, fundamentals, ratios, and earnings views.

## Version 4: Watchlists and Portfolio

Add user watchlists, portfolio tracking, holdings, profit and loss, and alerts.

## Version 5: Calendar and Events

Add IPO tracker, earnings calendar, RBI policy dates, and economic release calendar.

## Version 6: AI Market Intelligence

Add market summaries, movement explanations, company explainers, and news digestion.

## Version 7: Advanced Analytics

Add screeners, historical analysis, heatmaps, factor views, and machine-learning-assisted insights where useful.
